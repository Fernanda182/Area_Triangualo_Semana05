#  Programa para Calcular el Área de un Triángulo

Este proyecto consiste en un programa escrito en Python que permite calcular el área de un triángulo a partir de la base y la altura que introduce el usuario.

## Funcionalidades

- Solicita al usuario su nombre.
- Pide ingresar la base y la altura del triángulo (en centímetros).
- Calcula el área usando la fórmula:  
  \[ \text{Área} = \frac{\text{base} \times \text{altura}}{2} \]
- Muestra el resultado del área con dos decimales.
- Indica si el área es mayor a 20 cm² (`True` o `False`).

## Conceptos aplicados

- Tipos de datos: `str`, `float`, `bool`
- Funciones en Python
- Identificadores con convención `snake_case`
- Estructura de entrada y salida en consola
